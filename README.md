# My Cinema - Projet de gestion de cinéma

## Installation

1. Cloner le projet
2. Importer la base de données : `sudo mysql < script.sql`
3. Lancer le backend : `cd backend && php -S localhost:8000`
4. Lancer le frontend : `cd frontend && php -S localhost:3000`
5. Ouvrir http://localhost:3000/index.html

## Technologies

- PHP 8+ (POO, MVC)
- MySQL avec PDO
- HTML/CSS/JavaScript (Fetch API)
